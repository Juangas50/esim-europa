# Ruta34 Design Pack v1.0

Este pack organiza los mockups y referencias visuales de Ruta34 para usarlos en Claude Code, Cursor o cualquier flujo de implementación.

## Regla principal
Usar estos diseños como **referencia visual**, no como copia literal pixel-perfect. La implementación debe respetar la lógica actual del proyecto, precios, rutas, validaciones y flujo de compra.

## No hacer
- No mencionar Vodafone.
- No cambiar precios ni planes.
- No cambiar lógica de checkout.
- No introducir área privada ahora.
- No rediseñar por rediseñar.

## Orden recomendado
1. Design tokens y estilos globales.
2. Home V2.
3. Pricing.
4. Mobile.
5. Checkout.
6. Loader, email, guía y errores.
7. QA visual.
